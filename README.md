# Assignment 1: Building in the Full Stack

# What is the assignment

The website is focused on the country of **Georgia**, showcasing its geography, culture, and history in a clear and accessible way.

## What's included
- A semantic HTML structure using modern HTML elements
- Responsive styling with CSS, including a mobile-friendly layout
- Basic interactivity using JavaScript and DOM manipulation
- A simple contact form for user interaction

## Stuff used
- HTML
- CSS
- JavaScript 

## Running the Project
You can view the site by opening `index.html` in your browser.
